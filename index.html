HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements are represented by tags.
  • HTML elements can have attributes that provide additional information about the element.

CSS

  • CSS: Cascading Style Sheets
  • CSS is used to style HTML elements.
  • CSS can be written in a separate file or in the head of the HTML document.
  • There are three ways to add CSS to an HTML document: inline, internal, and external.
  • Inline CSS is added directly to the HTML element using the style attribute.
  • Internal CSS is added to the head of the HTML document using the style element.
  • External CSS is added to a separate file and linked to the HTML document using the link element.
  • There are three types of selectors: element, class, and id.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property. For example, the declaration color: blue; includes the CSS property color and the value for that property, blue.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add .: adds all files to the staging area
  • git commit -m "message": commits the files to the local repository
  • git push origin branch-name: pushes the files to the remote repository
  • git pull origin branch-name: pulls the files from the remote repository
  • git checkout branch-name: switches to the specified branch
  • git branch: lists all branches
  • git branch -d branch-name: deletes the specified branch
